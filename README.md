ISklep API client
=================

Based on [ISklep API docs](http://grzegorz.demos.i-sklep.pl/api_docs/)

##### Requirements

1. php ^5.6
2. valid isklep host
3. login
4. pass

##### Testing

Run:

```
composer install
```

Then:

```
vendor/bin/behat
vendor/bin/phpunit
```
